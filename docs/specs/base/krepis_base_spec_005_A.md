**발행일: 2025\. 12\. 31**

**상태: 최종 확정 (Finalized)**

**핵심 가치: 지능형 노동력 대체(AI Workforce), 초고성능 가속(Hyper-Latency), 95:5 상생 경제**

## **1\. 과금 체계 개요**

**Krepis는 \*\*Sovereign Kernel(Rust)\*\*의 물리적 제어권과 Fractal AI 군단의 지능 지수를 결합하여 과금 가치를 창출합니다. 성능(Latency) 뿐만 아니라, 투입되는 AI의 \*\*'추론 깊이'\*\*와 \*\*'자율성 수준'\*\*이 플랜의 핵심 차별점입니다.**

## **2\. 5단계 플랜 상세 명세 (The 5-Tier Sovereign)**

### **\[Tier 1\] FREE (Learners & Open Source)**

* **구독료: $0 / 월**  
* **지연 제어: 1.5\~2.5초 인위적 지연 (가장 낮은 스케줄링 우선순위).**  
* **AI 수준: RAG 기반 기본 가이드 (경량 보병 모델).**  
* **USE 기능: 사용 불가.**  
* **특이사항: 비식별 AST 데이터 전송 및 모델 학습 기여 동의 필수.**

### **\[Tier 2\] PLUS (Individual Developers) \- 신설**

* **구독료: $19 / 월**  
* **지연 제어: 0.8초 내외 (표준 응답).**  
* **AI 수준: 정예 보병 모델 (Qwen 7B / Llama 3B 최적화).**  
* **USE 기능: 월 1회 (소규모 단일 모듈 생성 한정).**  
* **비고: 개인 토이 프로젝트 및 단순 기능 구현 최적화.**

### **\[Tier 3\] PRO (Professional Architects)**

* **구독료: $59 / 월 (기존 $29에서 가치 기반 상향)**  
* **지연 제어: 0.4초 이내 (Turbo 모드 네이티브 가속).**  
* **AI 수준: 부사관급 군단 (Codestral 22B / Mistral Small).**  
* **USE 기능: 월 3회 (단일 마이크로서비스 및 API 전체 구축).**  
* **마켓플레이스: 수수료 5% 적용, 모듈 판매 및 Verified 인증 자격.**

### **\[Tier 4\] ULTRA (Startups & High-Growth Teams) \- 신설**

* **구독료: $149 / 월**  
* **지연 제어: 0.1초 (Hyper-Pingora 최우선 스케줄링).**  
* **AI 수준: 장교급 군단 (Mistral Large 2 / DeepSeek-V3).**  
* **USE 기능: 월 10회 (대규모 시스템 설계 및 연동 전체 구축).**  
* **특화 기능: Sovereign Sentry (자율 치유/자가 수정) 엔진 상시 가동.**  
* **지원: 우선순위 기술 지원 및 전용 API 레이턴시 보장.**

### **\[Tier 5\] ENTERPRISE (Sovereign Corporations)**

* **구독료: $999 / 월 (Base) \+ 사용량 기반 할증**  
* **지연 제어: 0초 (실시간 리스폰스, 전용 대역폭 점유).**  
* **AI 수준: 사령부급 (기업 전용 SFT 모델 \+ Air-gapped 인스턴스).**  
* **USE 기능: 무제한 (혹은 협의된 전용 리소스 내 자유 이용).**  
* **거버넌스: 사내 보안 규정(Compliance) AI 강제 주입, Private 레지스트리 제공.**

## **4\. 마켓플레이스 수익 배분 정책 (95:5 Economy)**

* **수익 배분: 제작자 95% : 플랫폼 5%.**  
* **AI 채택 인센티브: "USE" 기능이 마켓플레이스 모듈을 선택하여 서버를 구축할 경우, 해당 모듈 제작자에게 정산액의 1%를 \*\*'AI 적합성 보너스'\*\*로 추가 지급 (플랫폼 5% 중 일부 활용).**  
* **정산: 매월 15일 정산. USDC(Stablecoin) 정산 시 플랫폼 수수료 1% 추가 감면 프로모션 상시 진행.**

## **5\. 전략적 분석: 왜 이 가격인가?**

1. **지능형 노동력의 대체: $149(Ultra)는 시니어 개발자 1명의 일급보다 저렴하지만, 한 달간 수십 명의 개발자 분량의 설계를 무결점으로 뽑아냅니다. 사용자는 '도구'가 아닌 \*\*'인건비 절감'\*\*에 비용을 지불하게 됩니다.**  
2. **리소스 효율 극대화: 인위적 지연(Latency Shaping)과 모델 차등 배치를 통해 GPU 비용을 지능적으로 통제하여 플랫폼의 영업 이익률을 40% 이상으로 유지합니다.**  
3. **데이터 주권의 유료화: 보안 고립(Air-gap)은 단순 기능이 아니라 기업의 '생존' 문제입니다. 이를 Enterprise 플랜의 핵심 가치로 두어 높은 객단가(ARPU)를 확보합니다.**
