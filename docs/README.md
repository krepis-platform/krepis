## **Ⅰ. 프로젝트 비전 및 하이브리드 철학**

### **1\. 서비스 정의: ADaaS (Architecture Development as a Service)**

Krepis는 AI 군단과 인간 아키텍트가 협업하여 설계-생성-검증-유지보수를 자율 수행하는 **세계 최초의 프랙탈 기반 아키텍처 자율 주행 플랫폼**입니다.

### **2\. 하이브리드 엔진 철학: "Sovereign Control, Fractal Intelligence"**

* **Systemic Rigidity (Rust \+ Deno):** 핵심 엔진은 **Hyper-Pingora** 기반 Rust로 구축하고, **Deno**를 통해 런타임 결정성을 확보합니다.  
* **Fractal Intelligence (AI Legion):** 단일 모델의 한계를 넘어, **Master-Expert-Sub**로 이어지는 프랙탈 구조의 AI 군단이 무결점 코드를 생산합니다.  
* **Explicit Context Propagation:** **AsyncLocalStorage를 폐기**하고, Context 객체를 AI 지휘 체계의 신경망으로 활용하여 상위 의도를 말단 Atomic Executor까지 100% 전파합니다.

## **Ⅱ. 차세대 통신 및 빌드 표준**

### **1\. KNUL (Krepis Networking Ultra Link)**

* **Rust-Powered QUIC:** 0-RTT 핸드셰이크를 통한 지연 시간 극최소화(0\~10ms).  
* **Krepis-LZ:** 패턴 인식 기반 시맨틱 압축으로 페이로드 90% 절감.

### **2\. 하이브리드 통합 빌드 파이프라인**

* **Unified Simulation:** \*\*Standard(TS Mock)\*\*와 **Turbo(Rust Native)** 엔진을 동일 코드베이스에서 스위칭하여 개발 편의와 프로덕션 성능을 동시에 달성합니다.  
* **Recursive CI:** AI가 생성한 모든 Micro-Task 코드는 \*\*\[작성 → 네이티브 테스트 → 자가 수정\]\*\*의 무한 루프를 통과해야 커밋됩니다.

## **Ⅲ. AI 군단 아키텍처 (Fractal Intelligence)**

### **1\. 계층별 모델 배치 (Legion Hierarchy)**

* **LegionMaster (The Brain):** 전략 수립 및 Task 분할. (Llama 3.3 70B, Mistral Large 2\)  
* **SovereignArchitect (The Guard):** 도메인별(보안, 성능) 정밀 검수. (DeepSeek-V3, Mistral Small 3\)  
* **Atomic Executor (The Hands):** 원자 단위 코드 생성 및 무한 루프 검증. (Qwen2.5-Coder 7B, Codestral Mamba)

### **2\. "USE" 자율 구축 시스템**

사용자가 대화를 통해 명세서를 입력하면, AI 군단이 **무결점 서버를 통째로 구축**하고 가상 환경에서 성능을 증명한 뒤 배포합니다.
